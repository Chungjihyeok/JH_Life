### Git 명령어
```
1. commit 
2. branch
3. checkout
4. merge
5. reset
6. revert
7. cherry-pick
8. rebase 
```
#### commit 
<span style="background-color: #fdbb5d">커밋은 git 저장소에 여러분의 디렉토리에 있는 모든 파일에 대한 스냅샷을 기록하는 것입니다.</span>
<span style="background-color: #fdbb5d">각 커밋은 저장소의 이전 버전과 다음 버전의 변경내역을 저장합니다.</span>
<span style="background-color: #fdbb5d">그래서 대부분의 커밋이 그 커밋 위의 부모 커밋을 가리킵니다.</span>
<span style="background-color: #fdbb5d">쉽게 말해 어떠한 작업을 마치고 변경된 파일 내용을 작업 단위별로 관리하기 위해 기록하는 행위 입니다.</span>
